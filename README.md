這是一個專為**食物森林（Food Forest）**推廣而設計的開源遊戲化教材。透過互動式的放置經營模式，讓一般民眾與講座參與者能直觀地理解「七層結構」與「生態共生」的核心概念。

## 📖 專案緣起

食物森林不僅僅是種樹，它是一個模仿自然森林層次、能自我維持的永續耕作系統。然而，複雜的生態學常讓大眾卻步。本專案旨在透過**「數位遊戲化」**將枯燥的農業知識轉化為：

* **視覺化**的層次堆疊。
* **直覺化**的共生配對。
* **集體參與**的社區綠化體驗。

---

## 🎮 遊戲核心機制：【城市綠洲大亨】

玩家將從一塊「水泥荒地」開始，透過逐步解鎖不同層次的植物，建立一個完整的食物森林系統。

### 核心功能

* **垂直層次解鎖：** 必須先種植冠層樹（Canopy），才能解鎖下方受保護的耐陰植物。
* **夥伴共生系統 (Guilds)：** 根據植物屬性（如固氮、驅蟲、動態蓄積）獲得生長加成。
* **講座互動模式：** 支援多人在線即時投票，共同決定社區森林的樣貌。

## 📂 資料庫結構说明

本專案採用資料與邏輯分離的設計，方便開發者與環境教育推廣者進行擴充。

### 植物資料庫 (`/data/plants_data.json`)

目前已定義 **20 種台灣常見食物森林植物**，包含：

* **七層結構標籤：** 從層次 1 (冠層) 到層次 7 (攀援層)。
* **功能性屬性：** 包含光照需求、水分、生長速度。
* **共生加成 (Guild Bonus)：** 定義植物與環境或其他植物的互動邏輯。

## 🛠️ 如何開始使用？

### 1. 作為開發者

你可以透過 `fetch` 讀取 `plants_data.json` 並整合到你的遊戲引擎（如 Phaser.js, Unity 或簡單的 React 網頁）中。

```javascript
// 範例：讀取植物資料
fetch('./data/plants_data.json')
  .then(response => response.json())
  .then(data => console.log(data.plants));

```

### 2. 作為教育推廣者

* **植物圖鑑：** 直接利用 JSON 內容製作精美的植物功能卡牌。
* **互動教案：** 根據 `guild_bonus` 欄位設計拼圖遊戲，讓學生尋找「誰是誰的好鄰居」。
